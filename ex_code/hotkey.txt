////////////////// git 관련

git init: Git 저장소를 초기화합니다.
git add [파일명]: 파일을 Staging Area에 추가합니다.
git commit -m "[커밋 메시지]": Staging Area에 추가된 파일을 커밋하고 커밋 메시지를 작성합니다.
git push [원격 저장소명] [브랜치명]: 커밋한 내용을 원격 저장소에 업로드합니다.
git pull [원격 저장소명] [브랜치명]: 원격 저장소에서 최신 코드를 내려받습니다.
git clone [원격 저장소 URL]: 원격 저장소를 복제하여 로컬 저장소를 생성합니다.
git branch: 로컬 브랜치 목록을 확인합니다.
git checkout [브랜치명]: 해당 브랜치로 이동합니다.
git merge [브랜치명]: 다른 브랜치에서 변경된 내용을 현재 브랜치로 병합합니다.
git status: 로컬 저장소의 상태를 확인합니다.
git log: 이전의 기록 확인 (나기기: q)
